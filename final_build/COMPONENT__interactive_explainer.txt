# COMPONENT — Interactive Explainer

## Purpose

Explain the two-part T1D cure problem, modern cure research, the OSAN AI bionic pancreas concept, and the biohybrid roadmap in a concise investor-ready visual story.

## Inputs

- User stage selection through buttons, previous/next controls, arrow keys, Home/End, or URL hash.
- Static editorial copy and SVG artwork stored locally.

## Outputs

- One active story scene.
- Updated headline, evidence label, takeaway, progress, caption, and URL hash.
- Five primary-source links in the evidence section.
- Public experience: https://osangen.github.io/osan-bionic-pancreas/
- Public source: https://github.com/OsanGen/osan-bionic-pancreas

## State

- `activeStage`: integer from 0 through 4.
- Stable hashes: `problem`, `equation`, `landscape`, `product`, `roadmap`.

## Edge cases

- Unknown URL hashes resolve to stage 0.
- Previous/next movement is bounded to valid scenes.
- Reduced-motion users receive effectively immediate transitions.
- Narrow screens stack the narrative and artwork without horizontal page scrolling.
- Short desktop screens use the artwork's intrinsic 12:7 ratio instead of fixed scene heights, keeping the full primary story in the first viewport.
- JavaScript-disabled users receive an explicit notice.

## Dependencies

- Browser-native HTML, CSS, JavaScript, and SVG.
- Google Fonts for Poppins and Inter; system-font fallbacks are defined.
- No package runtime dependencies.
- Managed visual QA uses pinned Playwright 1.61.0 in GitHub Actions; it is not shipped to site visitors.
