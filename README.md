# The Path to a Bionic Pancreas

An OSAN-styled, five-scene visual thesis explaining the major scientific paths toward a functional and biological cure for type 1 diabetes—and how an AI-powered bionic pancreas can serve as the practical bridge.

**Live experience:** https://osangen.github.io/osan-bionic-pancreas/

## Experience

- Five keyboard- and touch-friendly story stages
- Responsive SVG diagrams at desktop, tablet, and mobile widths
- OSAN CogSec Pastel visual system: proof-first, one lead accent per scene, Poppins + Inter
- Public medical evidence register with regulatory and peer-reviewed sources
- Static GitHub Pages deployment with no runtime dependencies

## Local use

Open `index.html` directly, or serve the directory with any static server.

```sh
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173`.

## Validation

```sh
npm run check
```

## Important boundary

This is a research and product-concept visualization. It is not medical advice, an approved medical device, or a claim that a universal cure for type 1 diabetes is currently available.
